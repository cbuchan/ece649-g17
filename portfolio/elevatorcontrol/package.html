This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Utility files

Utility File
 Description
Utility.java This class provides some example utility classes that might be useful in more than one spot.
MessageDictionary.java
 This class defines constants for CAN IDs that are used throughout the simulator.
SampleDispatcherMonitor.java
 This example monitor shows how to use the RuntimeMonitor hooks to check for fast speed between floors.

 This monitor detects the number of stops where the car becomes overweight when the door are open.
